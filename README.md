#### Data
The data for this project is publicly available, and can be downloaded as a .csv file from [this link](https://data.cityofnewyork.us/Health/DOHMH-New-York-City-Restaurant-Inspection-Results/43nn-pn8j/about_data).  This data set is read in to our preprocessing code as '../data/NYC_Restaurant_Inspections.csv', but the data is not included here due to file size constraints.  The data on the NYC OpenData website is updated frequently with new restaurant inspection results.
